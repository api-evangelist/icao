# ICAO (icao)
The International Civil Aviation Organization (ICAO) is a specialized agency of the United Nations that codifies the principles and techniques of international air navigation and fosters the planning and development of international air transport. ICAO provides data services and digital standards for the global aviation community.

### ICAO API
